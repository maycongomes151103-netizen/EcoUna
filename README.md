## EcoUna


Minha proposta com este código foi desenvolver uma solução voltada para pessoas que ainda não têm conhecimento sobre reciclagem, ajudando-as a aprender como e onde descartar seus resíduos corretamente em locais adequados.


🌍 Contextualização e Impacto (ODS 12)
O projeto está alinhado ao ODS 12 (Consumo e Produção Responsáveis) da ONU, que visa transformar a forma como a sociedade produz e consome bens, reconhecendo que o modelo atual de desenvolvimento exerce pressão excessiva sobre os recursos naturais do planeta. Ele busca o equilíbrio entre o crescimento econômico e a preservação ambiental, garantindo recursos para as futuras gerações.


Escolhemos esse tema pela sua extrema importância para a sociedade brasileira, com o objetivo de conscientizar a população e contribuir diretamente para a melhoria do meio ambiente.


💡 Funcionalidades
Para isso, integrei um chatbot capaz de responder a qualquer dúvida sobre o tema e implementei um quiz interativo ao final da página web para fixar o aprendizado.


🚀 Como rodar o projeto
Assim que você baixar a pasta do projeto, por favor, instale a extensão Live Server (no VS Code) para renderizar e testar o código localmente.


🛠️ Tecnologias utilizadas

HTML: Linguagem de marcação para estruturar a página.


CSS: Utilizado para o design, estilização e organização visual dos elementos.


JavaScript: Responsável pela lógica do programa, funcionamento do chatbot/quiz e gerenciamento dos dados.
